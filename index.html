What the widget does

  1. On click, the widget GETs https://65.108.95.149/x402/health on the live Universal Control x402 provider.
  2. The provider returns HTTP 402 Payment Required with an x402 v2 challenge: payTo=0x302d13d6a1feE0b32eDf8B76786354cd69426fD7, maxAmountRequired=1000 (atomic USDC), network=eip155:8453, asset=0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913.
  3. Your x402-aware wallet (PayAI, Bankr, thirdweb, or any browser wallet with EIP-3009 support) signs a USDC transferWithAuthorization and broadcasts it on Base mainnet.
  4. The provider verifies the payment via the PayAI facilitator, delivers the response, and USDC settles to the configured payout address above in the same block.
