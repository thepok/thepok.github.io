# Drummer

Kid-friendly step sequencer drum machine (static webpage).

## How to use

- Go to `/drummer/` on your GitHub Pages site.
- Tap squares to turn steps on/off.
- Use **Add row** to add more drum lines, and change the sound per row.
- Set **Speed** in **beats per second**.
- Press **Start** (or press Space) to loop. Press again to stop.
- Use the **Recorder** panel to record your own sound and save it to **My Kit (Recorded)**.

## Notes

- Sounds are synthesized via Web Audio, and you can also record samples into **My Kit** (saved locally in your browser).
- Works as a static site (no build step).
